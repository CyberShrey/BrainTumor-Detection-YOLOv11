# Brain Tumor Detection using YOLOv11

## Project Overview
This project focuses on detecting brain tumors from medical images using the YOLOv11 object detection model.

## Dataset
- Brain Tumor Detection Dataset
- Annotated tumor regions
- Train, Validation, and Test splits

## Model
- YOLOv11 Small
- Trained using Roboflow and Google Colab

## Objective
To accurately detect and localize brain tumors in medical images.
